# travel-checklist-generator
Міні-сервіс для створення персонального списку речей у подорож

Опис ідеї:
Travel Checklist Generator — це міні-сервіс, який автоматично формує персональний список речей, які потрібно взяти у подорож.
Чекліст генерується на основі таких параметрів:
тривалість подорожі
країна / місто призначення
пора року (зима/літо/міжсезоння)
тип поїздки (пляжний відпочинок, міський туризм, бізнес-поїздка, похід)
специфічні умови (холод, спека, дощ, гори тощо)

Мета проєкту:
Створити простий інструмент для мандрівників.
Автоматизувати процес створення списку речей.
Показати вміння працювати з Node.js та базовою логікою.
Продемонструвати повний цикл створення ПЗ: від ідеї до прототипу.

Основні сценарії використання
1️. Користувач обирає параметри поїздки
(місто, сезон, кількість днів, тип подорожі)
2️. Система генерує персональний чекліст
Можуть бути включені категорії:
документи
одяг
гігієна
техніка
ліки
спеціальні речі для конкретного сценарію
3️. Користувач отримує фінальний список у вигляді JSON
(або текстовий список — залежно від реалізації прототипу)
